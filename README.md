# node-kibana

Just a node.js server to run [kibana](https://github.com/elasticsearch/kibana)

## Install and run


	# clone
	$ git clone https:#github.com/revington/node-kibana.git
	# cd to project
	$ cd node-kibana
	# install dependencies
	$ npm install
	# run
	$ ./server.js
	# open http://localhost:3000
